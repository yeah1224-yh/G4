## Scénario Principal

1- Page d'acceuil  

# Comparer des Cours 

1- L'étudiant clique sur le boutton Comparer Cours  
2- L'étudiant clique sur le slot pour choisir le cours  
3- Le système affiche la liste des cours  
4- L'étudiant choisis les deux Cours  
5- L'étudiant clique sur Comparer  
6- Le système compare les deux cours et affiche le resultat de la comparaison  
7- L'étudiant peut comparer encore d'autre cours ou Retourne à l'acceuil  
8- L'étudiant clique sur Retourne à l'acceuil  
9- Le système revient à la page d'acceuil  

# Consulter la liste des cours

1- L'étudiant clique sur Consulter la liste des cours  
2- Le Système affiche la liste de quelques cours ainsi que ses details ( description, id, nom, Periodes, Session, Préalable )  
3- L'étudiant peut rechercher un cours par le mot clé ou le code ( id )  
4- Le système affiche la liste de(s) cours rechercher par l'étudiant  
5- L'étudiant selectionne le cours souhaité  
6- Le système selectionne le cours selectionner  
7- L'étudiant peut consulter la liste des avis  
8- Le système affiche la liste des avis  
9- L'étudiant peut Filtrer les avis ()  
10- Le système affiche les avis filtrer  
11- L'étudiant clique sur Retourne à l'acceuil  
12- Le système revient à la page d'acceuil  

# Profil 

1- L'étudiant clique sur Profil  
2- Le système affiche les options disponibles pour la personnalisation des centres d'interêts  
3- L'étudiant peut personnaliser son profil puis cliquer sur enregistrer  
4- Le système sauvegarde les choix de l'étudiant  
5- L'étudiant peut modifier ses choix ou clique sur Quitter  
6- Le système revient à la page d'acceuil  

## Scénario alternatif

# Comparer des Cours 

5a.1 - Le système affiche, Veuillez selectionner les deux cours.  
5a.2 - Le système affiche, Veuillez selectinner un autre cours, impossible  
        de comparer les mêmes cours.     
5a.3 - Le scénario se termine.  

# Consulter la liste des cours 

3a.1 - Le système ne trouve pas le cours demander et affiche, Veuillez entrez un code correct  
       " Ex: iftxxxx ( x sont des chiffres ) ou par nom du cours chercher "  
3a.2 - Le scénario se termine  

# Profil

3a.1 - L'étudiant ne clique pas sur enregistrer et essaie de quitter, le système affiche,  
       voulez vous enregistrer "Oui" ou "Non"  
3a.2 - Si Oui, le système sauvegarde ses choix, puis revenir à la paga d'acceuil  
3a.3 - Si Non, le système revient à la page d'acceuil sans sauvegarder ses choix
